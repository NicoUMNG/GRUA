🚗 Control de Motores ESP32

⚠️ Atención:
GitHub Pages usa HTTPS y no puede conectarse directamente a dispositivos HTTP locales como el ESP32.
💡 Solución:
Debes acceder directamente al ESP32 desde tu red local.

📋 Instrucciones:

  1. Asegúrate de que el ESP32 esté encendido y conectado a tu WiFi
  2. Verifica la IP del ESP32 en el Monitor Serie de Arduino
  3. Ingresa la IP en el campo de arriba
  4. Presiona el botón para ir al panel de control
  5. Si no funciona, copia la IP y pégala directamente en tu navegador

🔧 Alternativa:

Si tienes problemas, puedes cargar el código modificado que incluye el HTML directamente en el ESP32 (sin necesidad de GitHub Pages).
